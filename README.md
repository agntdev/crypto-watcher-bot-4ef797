# Crypto Watcher Bot

Personal Telegram bot for private crypto watchlists, on-demand prices, and configurable alerts (thresholds, percent moves, quiet hours, morning summaries).

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
